# SpringBoot + Mysql + Spring Data JPA를 활용한 회원 프로젝트 

# 개발환경
- Open JDK 11
- intelliJ IDEA Community
- MySQL Server 8.1.0
- Spring Boot 2.7.18

# 라이브러리 , 프레임워크
-  Spring Data JPA
-  Thymeleaf
-  lombok
-  gradle

# 주요 기능 
- 회원가입
- 로그인
- 회원목록
- 회원조회
- 회원정보 수정
- 회원삭제
- 로그아웃
